Name of the project: Model of Credit Card Clients Default Prediction
Гsing Dataset:
      Default of Credit Card Clients Dataset from Kaggle.com
      (Default Payments of Credit Card Clients in Taiwan from 2005)
Goal: To determine the probability of Default in the next month for a customer of a bank, 
      according to his sociographic data and payment and debt history

How to run it:
1. Download dataset from https://www.kaggle.com/datasets/uciml/default-of-credit-card-clients-dataset/data
2. Open notebook Hackton_ML.ipynb in Google.Colab
3. Execute each code cell in sequence
